# Personal Expense Tracker 

My first Python project dedicated to managing and tracking personal daily expenses.

## 📖 Description
This is a Command-Line Interface (CLI) application designed to help users record and monitor their spending habits. The project is built using **Object-Oriented Programming (OOP)** principles to ensure clean, organized, and scalable code.

## ✨ Features
- **Object-Oriented Design:** Uses Python Classes to manage expense items efficiently.
- **Input Validation:** Implements `try-except` blocks to handle price entry errors and ensure data integrity.
- **Categorization:** Allows users to categorize expenses (e.g., Food, Transport, Study).
- **Automated Summary:** Instantly calculates the total expenditure and displays a detailed summary of all entries.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Development Environment:** Linux (Ubuntu)
- **Version Control:** Git & GitHub
